# Map-Odom-Base-link

In ros2 navigation system,'map','odom',and 'base_link'are three important coordinate frames.
They form the core structure of the TF Tree used for localization and navigation.

## Map Frame

It represents the fixed world map generated by SLAM or loaded by a saved map.
It is a fundamental frame.

## Odom Frame

It represents the robot's local motion over time.

It is usually generated from wheel odometry or other motion sensor.
This frame changes continously as the robot moves.
At the same time,odometry also accumulate errors overtime,so it needs the data in map and adjust.

## Base_link Frame

It represents the robot body frame.

It is attached to the robot itself and moves together with the robot.
