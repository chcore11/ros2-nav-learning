# ROADMAP

This project aims to show a half-year learning plan for ros2-nav-learning,if you take my advise and use tools skillfully,you can absolutely do a good job.

## Learning Stages
 
### 1:Install WSL2,Ubuntu,ROS2,Bazebo,RViz2(You can also try to manage a github account if you wanna extend or makes your work visable)1

### 2:Understand how a robot precerives and simulation by launching a Turtlebot3 in Gazebo,and create a teleop to control it.(Understand tf-tree!)

### 3:Run SLAM with maps you made,learn how robots build maps of unkonwn environment

### 4:Let robots to determine its position on a known map with the help of AMCL and RViz2

### 5:Implement autonomous goal-directed navigation

### 6:Deepen understanding of the hole navigation system
