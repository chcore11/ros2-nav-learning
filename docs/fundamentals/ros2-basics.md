# ros2-basics

ROS2(Roboyt Operating System 2) is a middleware framework designed for building modular and distributed robotic systems.

## Core Concepts

### Node
An independent process that performs a specific task in a robot system,such as sensor processing and motion control.

### Topic
Topics provides a publish-subscribe communication mechanism between nodes.

### Message
It defines the data structure transmitted through topics.

### Service
Services implement synchronous request–response communication between nodes.

### Action
Actions are used for long-running tasks such as navigation goals.
