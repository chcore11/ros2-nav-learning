# Map Server and Static Map

After a map is created using SLAM,it can be saved and reused later.
In ros2 navigation systems.this saved map is called a "static map"

## Static Map

A static map is a previously generated map of the environment.
Once saved,this map can be reused without running SLAM again.

## Map Server

The map server loads the saved maps and publishes it to the ros2 system.

## Role in This Project

In this project,the map server is used to load the map generated by SLAM.

This allows the robot to navigate in a known environment without rebuilding the map.
