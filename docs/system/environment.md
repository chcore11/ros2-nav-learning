# Environment

This project runs a ROS2 Navigation simulation enviroment on a Windows laptop using WSL2

## System Environment
 - Host OS:Windows
 - Linux Environment:WSL2 with Ubuntu
 - Unnescessary but recommend:VScode,Git,Github

## Softwares
 - ROS2:robot operarting system
 - Gazebo:environment for robot simulation
 - Rviz2:visulization tool
  
## Navigation Components
 - SLAM(Simultaneous Localization And Mapping)
 - Nav2:navigation framework
 - AMCL:localization
